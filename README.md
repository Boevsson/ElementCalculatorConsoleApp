# ElementCalculatorConsoleApp
C# ElementCalculatorConsoleApp

Enter a molecule structure and you will get a string of the total count of every item in the molecule.

Example:

Input:                  Output:
"NaCl"                  "Cl1Na1" 
"Ca(OH)2                "Ca1H2O2"
"Al2(SO4)3"             "Al2O12S3"
"[Cu(NH3)4(H2O)2]SO4"   "Cu1H16N4O6S1"
